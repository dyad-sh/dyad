Dyad is a free, local AI app builder for power users. Build full-stack web applications by chatting with AI—no coding required. Unlike cloud-based alternatives like Lovable, v0, or Bolt, Dyad runs entirely on your machine, giving you complete privacy and control.

[![Dyad Screenshot](https://github.com/user-attachments/assets/f6c83dfc-6ffd-4d32-93dd-4b9c46d17790)](https://dyad.sh/)

## Why Dyad?

| Cloud-based builders             | Dyad                                                                  |
| -------------------------------- | --------------------------------------------------------------------- |
| Your code lives on their servers | **Your code stays on your machine**                                   |
| Locked into their AI model       | **Bring your own API keys** (OpenAI, Anthropic, Google, local models) |
| Monthly subscriptions add up     | **Free and open-source**                                              |
| Limited export options           | **Full ownership—edit in any IDE**                                    |
| Internet required                | **Works offline with local models**                                   |

## Features

### Build Apps with AI

- **Multi-turn conversations** — Iteratively refine your app through natural dialogue
- **Real-time preview** — See changes instantly as you build
- **Smart context** — Intelligent file selection for optimal code generation
- **Auto-fix errors** — Automatic TypeScript error detection and correction

### Multi-Model Support

Use your own API keys with any major AI provider:

- **OpenAI** — GPT-4o, GPT-4 Turbo, o1, o3
- **Anthropic** — Claude Sonnet 4, Claude Sonnet 3.5
- **Google** — Gemini 2.5 Pro, Gemini 2.5 Flash
- **Local Models** — Ollama, LM Studio for complete privacy
- **Enterprise** — Azure OpenAI, Amazon Bedrock, Google Vertex AI

### Developer-Friendly

- **Git integration** — Built-in version control with branch management
- **Database support** — SQLite locally, or connect to Supabase/Neon
- **One-click deploy** — Ship to Vercel directly from Dyad
- **MCP support** — Extend with Model Context Protocol tools
- **NPM packages** — Add and manage dependencies seamlessly

### Cross-Platform

Works on **macOS**, **Windows**, and **Linux**.
