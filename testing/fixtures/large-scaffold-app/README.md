# Large Scaffold Fixture App

This app simulates a large application produced from Dyad's default React scaffold. It is intentionally broad and repetitive so agent tooling, code exploration, search, and context selection have a realistic project to inspect.

The app is a static operations command center. It uses Vite, React, TypeScript, React Router, Tailwind, shadcn-style UI components, TanStack Query, and deterministic mock data. There are no network calls, secrets, databases, or environment requirements.

## Commands

- `pnpm install`
- `pnpm dev`
- `pnpm build`
- `pnpm lint`
- `pnpm test`
